### 23강 - 브랜치 충돌 해결 실습

- 일반적인 머지 충돌 경우 1

```
<<<<<<<<< HEAD
....               <- 현재 체크아웃 한 브랜치의 소스코드
==============
....               <- 병합하려는 브랜치의 소스코드
>>>>>>>>> 병합하려는 브랜치 이름
```

- 일반적인 머지 충돌 경우 2
```
<<<<<<<<< HEAD
....               <- 현재 체크아웃 한 브랜치의 소스코드
|||||| merged common ancestors
....               <- 두 브랜치의 공통 조상의 소스코드
==============
....               <- 병합하려는 브랜치의 소스코드
>>>>>>>>> 병합하려는 브랜치 이름
```

- 해결방법
사용하고 싶은 코드 (`....`)만 남겨두고 나머지는 지운 후 저장하게 되면 정상적으로 저장이 되는데, `git status` 명령어로 추후의 행동에 대해서 도움을 받으면 된다.
